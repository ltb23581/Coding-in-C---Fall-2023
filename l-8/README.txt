4. init(), draw(), move(), won()

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
To compile the program, be in the directory Bui-Lindsay-p1

Type $ make to compile and $ make clean to remove all executables and object codes

Type $ fifteen [dimension value] to run the program. 
For example, $ fifteen 3 for a 3x3 dimension

You may also type $ ./fifteen [dimension value] 



To run directly to the winning configurations, type:

$ ./fifteen 3 < 3x3.txt
OR
$ fifteen 3 < 3x3.txt

Likewise for a 4x4 dimension
